# Заголовок
Настроить публикацию проекта в GitHub через консоль

# Пояснения
Git это система управлением версиями программы

GitHub удаленный хост хранения git систем

# Настройка
Скачать Git Bash Portable, распаковать

Создать папку “repo” внутри GitBash

Запустить git-bash.exe

Выполнить команду:
-	git config --global user.name “сюда” (ваш ник в гитхабе)
-	git config --global user.email сюда (ваш емеил в гитхабе)
-	git config --list (проверка)

-	ls (выводит список файлов)
-	cd repo (вход в папку repo)

Создать репозиторий в гитхабе, скопировать https ссылку

-	git clone https://github/zfjijicll/test.git (ваш https; копирует репозиторий с гитхаба на ваш ПК)
-	cd test (зайти в созданный реп.)

Создаем в папке repo/test любой файл, new file.txt

-	git status (видим новый файл)
-	git add . (добавляем файл к ветке мастер)
-	git status (видим что он добавился)
-	git commit (фиксируем изменения)

Видим вим редактор, вверху пишем коммент, что нового добавили.

i клавиша для редактирования 

esc конец редактирования

ZZ сохранение и отправка 

-	git remote --v (список удаленно подключенных хостов; стандартное имя origin)
-	git push origin master (спихнем на удаленный хост ветку мастер)

Выскочит гитхаб логин, все вести верно.
Если все верно, наша локальная версия спихнулась на удаленный хост
